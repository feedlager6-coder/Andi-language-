# Аудиофайлы слов андийского языка

## Статус
АУДИОЗАПИСЕЙ НЕТ. Эта директория подготовлена для будущих файлов.

## Формат файлов
- Имя файла: `{word_id}.mp3` (ID из таблицы `words` базы данных)
- Формат: MP3
- Параметры: моно, 44100 Hz, битрейт 128 kbps
- Нормировка: -3 dBFS peak

## Как получить ID слова
```bash
curl http://localhost:3001/api/words?limit=1000 | python3 -c "
import sys, json
for w in json.load(sys.stdin)['words']:
    print(f'{w[\"id\"]}: {w[\"andiWord\"]} — {w[\"russian\"]}')
"
```

## Кто может записать
- Носители андийского языка (с. Анди, Ботлихский район, Дагестан)
- ИЯЛИ ДНЦ РАН (Махачкала) — контакт через издателей словаря Салимова
- МГУ, кафедра языков народов России

## Рекомендации по записи
1. Тихое помещение (без эха)
2. Микрофон: студийный конденсаторный или петличный
3. Каждое слово — отдельный файл
4. Читать в нормальном темпе, без ускорения
5. Пауза 0.3 сек до и после слова
6. После записи — нормировать уровень

## Когда файлы загружены
Компонент в словаре автоматически покажет кнопку воспроизведения,
если файл `/audio/words/{id}.mp3` существует.
